# georgedelair.com

Personal website for George DeLair — web developer & automation consultant.

## Overview

A minimal, editorial-style personal website built with vanilla HTML and CSS. No JavaScript required for core functionality.

```

├── styles.css          # All styles

```

## Hosting

This site is designed for GitHub Pages or any static hosting. No build step required.

### GitHub Pages Setup

1. Push to a GitHub repository
2. Go to **Settings → Pages**
3. Select branch (usually `main`) and root folder
4. Your site will be live at `https://username.github.io/repo-name/` or your custom domain

### Custom Domain

To use a custom domain:
1. Add a `CNAME` file with your domain (e.g., `georgedelair.com`)
2. Configure DNS with your registrar

## Development

Just open `index.html` in a browser. No build tools or dependencies.
